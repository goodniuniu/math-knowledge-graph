函数应用（一）· 交互式学习

拖动参数 → 观察图像变化 → 看到计算结果 → 理解定义域限制

手机流量套餐计费（一次函数模型）
某运营商套餐：月租 b 元，超出赠送额度后每使用 1 GB 流量收费 k 元。 设当月使用流量为 x GB，则总费用 y = kx + b（简化线性模型）。
y = k·x + b
当前费用 y = 65.0 元
⚠️
定义域提醒：实际中 x ≥ 0（流量非负），且套餐通常封顶 100 GB（超出断网），故合理定义域为 0 ≤ x ≤ 100。另外若按整 GB 计费，x 应取非负整数，图像为离散点而非连续直线。
商品利润最大化（二次函数模型）
某商品每件进价为 m 元。当售价定为 x 元时，每天销量为 n − p·x 件（价格越高销量越低）。 每日利润 y = (x − m)(n − p·x)，是开口向下的二次函数，存在最大利润。
y = (x − m)(n − p·x) = −p·x² + (pm + n)·x − mn
最优定价 x* = 20.0 元
最大利润 ymax = 1000 元
当前利润 = 1000 元
配方法推导（求最值）
y = −p·x² + (pm + n)·x − mn
  = −p[ x² − pm + npx ] − mn
  = −p[ (x − pm + n2p)² − (pm + n2p)² ] − mn
  = −p(x − x*)² + (n − pm)²4p
∴ 当 x* = (pm + n)/(2p) = (n/p + m)/2 时，利润取得最大值 ymax = (n − pm)²/(4p)。图像顶点即最优点。
均值不等式的替代思路
利润 y = (x − m)(n − p·x) 可改写为 y = p·(x − m)·(np − x)。
令 u = x − m，v = np − x，则 u + v = np − m（为定值）。
由均值不等式，当 u = v 时乘积 uv 最大，即 x − m = np − x → x = (m + n/p)/2 = x*，与配方法结果一致。
⚠️
定义域提醒：售价必须不低于进价才不亏本 → x ≥ m；同时销量不能为负 → n − p·x ≥ 0 ⇒ x ≤ n/p。故合理定义域为 m ≤ x ≤ n/p。若定价超出此区间，模型失去实际意义。
居民阶梯水费（分段函数模型）
某地居民水费实行阶梯计价：用水量 x（吨）在 0–12 吨部分单价 a 元/吨； 12–18 吨部分单价 b 元/吨；超过 18 吨部分单价 c 元/吨。总费用 y 由三段拼接而成。
y = ⎧a·x ,  0 ≤ x ≤ 12
    12a + b(x−12) ,  12 < x ≤ 18
    12a + 6b + c(x−18) ,  x > 18
当前水费 y = 35.0 元
所在阶梯：第二阶梯
⚠️
边界值提醒：分段函数在 x = 12 和 x = 18 处函数值连续（前段末值 = 后段初值）， 但单价（斜率）发生跳跃——这是"断点"的真正含义。计算时务必先用 x 判断所属区间，再代入对应解析式， 切勿全程用单一单价。定义域：x ≥ 0。
函数应用题五步解题法

点击每一步展开详细说明与示例。最后一步"还原实际意义"最易被忽视，却决定答案是否正确。

1审题点击展开
仔细阅读题目，弄清已知量与未知量，识别变量之间的依赖关系，判断适合用哪类函数模型（一次 / 二次 / 分段）。
示例：题目说"每多生产 1 件，成本增加固定数额"→ 线性关系；"围成矩形，面积随边长变化"→ 二次关系；"不同区间单价不同"→ 分段关系。
2设未知数点击展开
选取合适的自变量（通常用 x 表示"可变化的量"，如价格、用量、边长），因变量 y 表示要求的目标量（费用、利润、面积等）。
示例：设售价为 x 元/件，则每日利润为 y 元。设用水量为 x 吨，则水费为 y 元。
3建立函数关系点击展开
根据题意列等式，写出 y 关于 x 的解析式，并注明定义域（受实际背景限制）。
示例：利润 y = (x − m)(n − p·x)，且 x ≥ m、x ≤ n/p；水费 y 为三段拼接，x ≥ 0。
4求解点击展开
利用函数性质求解：一次函数看单调性；二次函数用配方法或顶点公式求最值；分段函数分段计算后比较。
示例：二次函数在顶点 x* = (pm+n)/(2p) 处取最大值；分段函数需逐段代入再合并。
5还原实际意义⭐ 最重要
把求出的数学结果翻译回实际语境，检验是否在定义域内、是否符合常理，并按题目要求取整或表述。 这一步能发现"负值""超界""应取整数"等隐藏错误，是得分的关键。
示例：算出最优定价 x* = 20.3 元，但实际标价只到角 → 取 20.3 元或比较 20 与 20.5 取更优者； 算出利润为负 → 说明此定价亏本，应舍去；面积问题中边长不能为负。
⚠️ 常见失分：只写出"x = 20.3"就结束，却没说明"定价约 20.3 元时利润最大"； 或求出两个数学解却未剔除不符合实际背景的那个。务必"还原"！
